## Gemini API Setup (Google Generative Language)

1. Create a `.env` file in the project root (there is a template `.env` included).
2. Set `GEMINI_API_KEY` to your Google Cloud API key for the Generative Language API.
3. Run the app:

```bash
flutter pub get
flutter run
```

The app will use Gemini (via the Generative Language API) to summarize text when you trigger the summarize action.
